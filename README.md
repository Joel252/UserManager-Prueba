# UserManagerPrueba


Este proyecto es una aplicación web desarrollada en **ASP.NET Core MVC** con **Identity** para la autenticación de usuarios. Permite crear, editar, eliminar y gestionar usuarios desde una interfaz de administración (CRUD).

### Características:
- Registro de usuarios
- Autenticación mediante **ASP.NET Identity**
- CRUD de usuarios (Crear, Leer, Actualizar, Eliminar)

## Requisitos previos

Antes de ejecutar la aplicación, asegúrate de tener instalados los siguientes requisitos en tu máquina:

- **.NET SDK** (versión 8 o superior)
- **Visual Studio** o **Visual Studio Code** (con extensión de C#)
- **SQL Server** o **SQLite** (dependiendo de la configuración de la base de datos)

## Instalación

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/tuusuario/tu-repositorio.git
   ```

2. Restaurar dependencias
Si usas Visual Studio, las dependencias se restauran automáticamente al abrir el proyecto. Si usas la terminal, ejecuta:

```bash
dotnet restore
```

3. Ejecutar el proyecto
Para compilar y ejecutar la aplicación, ejecuta el siguiente comando en la terminal o simplemente presiona Iniciar (F5) en Visual Studio:

dotnet run
